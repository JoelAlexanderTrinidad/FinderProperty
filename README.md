### Información General
***
Sitio para buscar/crear propiedades. Creado con:
  * Backend
    - NodeJS
    - ExpressJs
    - MySql
    - ORM - Sequelize - sequelize-cli
  * Frontend
    - Vite - ReactJS
    - Tailwind - CSS
    - Autenticación y Autorización con Firebase
